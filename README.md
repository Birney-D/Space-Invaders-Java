# Space Invaders (Java)

A Java-based arcade shooter inspired by the classic **Space Invaders** game. This project features real-time keyboard controls, enemy waves, barricades, sound effects, and a GUI-based game loop. This was one my very first personal projects so its less than perfect...please keep that in mind before you deal out the harsh criticism ;)

## Features
- Player movement and shooting
- Enemy wave movement and attacks
- Defensive barricades
- Collision detection
- Sound effects
- GUI-based gameplay

## Technologies Used
- Java
- Maven
- Swing / AWT

## Project Structure
- `src/main/java/spaceinvaders/` – game source code
- `resources/` – game images and assets
- `pom.xml` – Maven build configuration

## How to Run:
```bash
    mvn clean package
    java -jar target/<yourjarfile.jar>
```
